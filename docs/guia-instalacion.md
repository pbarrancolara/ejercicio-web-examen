# Guía instalación de mi aplicación

---

### Paso 1
Para empezar tienes que hacer un ``npm init -y`` para inicializar el proyecto, le instalas las dependencias necesarias que en mi caso es jest y jsdoc entonces tendrias que hacer un ``npm intall jest`` y ``npm install jsdoc``.

## Paso 2 
Ahora ya podrias entrar en programar tu aplicación, te creas tu html, css y tu js bien documentado para hacer la documentación automática.

## Paso 3
Despues de tener tu aplicación programada , tendrias que programas unas pruebas unitarias para comprobar que las funciones de javascript esten bien y no tengan fallos.

## Paso 4 
Por ultimo debes haber documentado todo este proceso, desde las tecnologias que vas a utilizar (JavaScript, HTML, CSS, etc), tanto como las herramientas (JSDOC, jest);
